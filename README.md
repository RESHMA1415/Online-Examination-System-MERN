# Online Examination System (MERN Stack)

##  Project Description

The Online Examination System is a MERN Stack web application developed for conducting online examinations. It provides separate modules for Students and Administrators.

---

## Features

### Student Module

- Student Registration
- Student Login
- Dashboard
- Online Examination
- Automatic Score Calculation
- Pass / Fail Result

### Admin Module

- Admin Login
- Admin Dashboard
- Add Questions
- Edit Questions
- Delete Questions
- View Students
- View Results

---

## Technology Stack

### Frontend

- React.js

- Axios

### Backend

- Node.js

##  Installation

### Clone Repository

```bash
git clone https://github.com/RESHMA1415/Online-Examination-System-MERN.git
```

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev
```
